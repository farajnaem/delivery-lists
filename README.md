# كشوفات التسليم

نظام تجهيز كشوف توزيع الطرود الإغاثية: استيراد Excel، توزيع على أيام وشبابيك، وتصدير Excel منسّق.

## التشغيل المحلي

```powershell
cd Desktop\كشوفات التسليم
copy .env.example .env
composer install
php database\install.php
php -S localhost:8090 -t public public/router.php
```

أو: `start-local.bat`

افتح: http://localhost:8090/setup — أنشئ حساب المدير.

## سير العمل

1. **عملية توزيع جديدة** — رفع Excel + بيانات الطرد والمخزن
2. **توليد الكشوف** — توزيع تلقائي (أيام → شبابيك → ساعات 09:00–15:00)
3. **تنزيل Excel** — أوراق: الإجمالي، تسليم يوم/شباك، الرسائل

## Excel المستورد

| العمود في Excel | مطلوب | ملاحظة |
|-----------------|--------|--------|
| **اسم رب الأسرة** | ✅ | أو: اسم المستفيد، الاسم |
| **رقم الهوية** | ✅ | |
| **رقم التواصل** | ✅ | أو: رقم الجوال |
| **حالة الاستلام** | اختياري | افتراضي: قيد التسليم |
| مركز الإيواء | ❌ | يُتجاهل حالياً |
| تاريخ الترشيح | ❌ | يُتجاهل حالياً |

## الأدوار

- **مدير النظام** — كل الصلاحيات + المستخدمون
- **منسّق التوزيع** — إنشاء وتوليد وتصدير
- **مراجع** — معاينة + تصدير
- **مشاهد** — معاينة فقط
- **أمين المخزن** — تسليم من الجوال (`/warehouse`) أو **تطبيق أندرويد** (`android-app/`)
