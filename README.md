# كشوفات التسليم

نظام تجهيز كشوف توزيع الطرود الإغاثية: استيراد Excel، توزيع على أيام وشبابيك، وتصدير Excel منسّق.

## التشغيل المحلي

```powershell
cd Desktop\كشوفات التسليم
copy .env.example .env
composer install
php database\install.php
php -S localhost:8090 -t public public/router.php
```

أو: `start-local.bat`

افتح: http://localhost:8090/setup — أنشئ حساب المدير.

## سير العمل

1. **عملية توزيع جديدة** — رفع Excel + بيانات الطرد والمخزن
2. **توليد الكشوف** — توزيع تلقائي (أيام → شبابيك → ساعات 09:00–15:00)
3. **تنزيل Excel** — الكلي: الإجمالي + كشوف التسليم؛ الرسائل وكشوف يوم معيّن من أزرار «تنزيل يوم بيوم»

## كود الصرف

- الصيغة: `كود الطرد` + `7 أرقام` عشوائية غير متسلسلة — مثال: `SOCI4829103`
- كل مستفيد في الطرد يحصل على **كود صرف فريد** (رقم عشوائي من 7 خانات) لا يتكرر حتى انتهاء الطرد
- قاعدة البيانات تمنع تكرار `disbursement_code` داخل نفس العملية
- في كشوف التسليم والاستلام والتقارير: الكود كامل بالحروف والأرقام
- في كشوف الرسائل وتسليم المخزن: الرقم فقط (`4829103`) **بالأرقام الإنجليزية**
- **العرض في النظام**: الأرقام العربية `٠١٢٣` والوقت بنظام **12 ساعة** (`٢:٣٠ م`) — **الرسائل SMS بالأرقام الإنجليزية**
- الترتيب داخل كل كشف: أبجدي تصاعدي حسب الكود (مع الإبقاء على توزيع الأيام/الشبابيك/الساعات)

## كشوف الرسائل

- التصدير **يوم بيوم** من صفحة العملية (رسائل يوم 1، يوم 2، …) لتقليل خطر الطوارئ/التأجيل
- **رسائل_جوال**: الأرقام التي تبدأ بـ `59`
- **رسائل_أوريدو**: الأرقام التي تبدأ بـ `56` مع إضافة `972` قبل الرقم في الكشف
- **رسائل_غير_مصنفة**: أي رقم لا يبدأ بـ `59` أو `56` (للمراجعة فقط)

## كشوف التسليم

- التوليد مرة واحدة لكل الأيام؛ يمكن تنزيل كشوف يوم معيّن للطباعة من صفحة العملية
- Excel الكامل: الكشف الإجمالي + كل أوراق يوم/شباك (بدون رسائل — الرسائل منفصلة يوم بيوم)
