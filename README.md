# كشوفات التسليم

نظام تجهيز كشوف توزيع الطرود الإغاثية: استيراد Excel، توزيع على أيام وشبابيك، وتصدير Excel منسّق.

## التشغيل المحلي

```powershell
cd Desktop\كشوفات التسليم
copy .env.example .env
composer install
php database\install.php
php -S localhost:8090 -t public public/router.php
```

أو: `start-local.bat`

افتح: http://localhost:8090/setup — أنشئ حساب المدير.

## سير العمل

1. **عملية توزيع جديدة** — رفع Excel + بيانات الطرد والمخزن
2. **توليد الكشوف** — توزيع تلقائي (أيام → شبابيك → ساعات 09:00–15:00)
3. **تنزيل Excel** — أوراق: الإجمالي، تسليم يوم/شباك، رسائل جوال، رسائل أوريدو

## الأدوار

- **مدير النظام** — كل الصلاحيات + المستخدمون
- **منسّق التوزيع** — إنشاء وتوليد وتصدير

## كود الصرف

- داخلي: `SOCIR2648291` (SOCI + ملحق + رقم عشوائي)
- للمستفيد والشبابيك والمخزن: رقم فقط بدون أصفار يسار (`48291` أو `482`)
- في تقارير التسليم: الملحق + الرقم (`R2648291`)
- في Excel: خانة الكود تُصدَّر كرقم وليس نص

## كشوف الرسائل

- **رسائل_جوال**: الأرقام التي تبدأ بـ `59`
- **رسائل_أوريدو**: الأرقام التي تبدأ بـ `56` مع إضافة `972` قبل الرقم في الكشف
- **رسائل_غير_مصنفة**: أي رقم لا يبدأ بـ `59` أو `56` (للمراجعة فقط)
